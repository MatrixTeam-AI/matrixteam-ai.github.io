# MAMBO-G Project Page

Static project page for:

**MAMBO-G: Magnitude-Aware Mitigation for Boosted Guidance**

## Structure

```text
project-page/
├── index.html
├── static/
│   ├── css/main.css
│   ├── images/
│   │   ├── figs/
│   │   └── logos/
│   └── paper/mambo-g.pdf
```

## Local Preview

The page is static and can be opened directly:

```text
Mambo-g/project-page/index.html
```

For a localhost preview:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000/Mambo-g/project-page/
```
